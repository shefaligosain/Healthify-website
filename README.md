# Healthify-website
This project is a fully functional website that looks fantastic on any computer, tablet, or smartphone. Each page you visit while using the website is fully responsive and nicely designed. The content of the page is fully descriptive and informative making it easier for the user to interact with it. The recipes mentioned in the website are along with features like-procedure, cooking time, ingredients needed and how many calories it contains. In addition to this, it also contains a navigation bar which would facilitate the user/client to further navigate distinct aspects of the page. All this adds up to an alluring website which may contain more features and recipes so that it is fully ready to launch in future.
